les sélecteur css

  • une balise : h1 {... }
  • une classe : .ma_classe {...}
  • un identifiant : #mon_id {...}
  • un attribut : [data-attr] {...}
  • une combinaison : ul li {...} ou ul > li {...} ou p + p {...}

Box Model

  • Le "Modèle de boite" en CSS définit si un élément est un BLOC ou "en ligne".
    • Un BLOC comporte une largeur, une hauteur, des marges intérieures et extérieures, des bordures, etc.
    • Un élément EN LIGNE s'insère dans le reste du contenu, ne comporte pas de largeur définie
    • cf. là
